## Quick Start
1. Install dependencies:
   pip install -r requirements.txt

2. Set license variables (Windows):
   set CYBERDUDEBIVASH_LICENSE_KEY=your-key
   set CYBERDUDEBIVASH_EXPIRY_DATE=2027-01-29
   set CYBERDUDEBIVASH_HARDWARE_HASH=your-hash

3. Scan:
   python cyberdudebivash-op-model-sentry.py model.pth
   or
   python cyberdudebivash-op-model-sentry.py --dir models/ --recursive
